# 안녕하세요, 프론트엔드 개발자 배상준입니다.

웹 · 데스크탑(Electron) · WebView(Flutter) 등 다양한 환경에서  
안정적으로 동작하는 프론트엔드 아키텍처를 설계하고 구축합니다.

Next.js 기반 웹서비스, 인증 구조(httpOnly 쿠키),  
스크래핑 자동화(Electron + Puppeteer), UI/UX 리팩토링(dnd-kit) 등을 경험하며  
복잡한 문제를 단순한 구조로 만드는 것을 중요하게 생각합니다.
